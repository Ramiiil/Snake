# Snake
Python game
Классическая игра змейка. Кушайте яблоки и наслаждайтесь музыкой
Чтобы начать игру запустите код. 
Управление змеи осуществляется стрелками и wasd. 
Чтобы завершить, закройте окно или нажмите кнопку escape
